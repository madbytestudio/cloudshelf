Features

Everything you need for a secure, private reading experience.

Offline PDF Reading

Download and read your PDFs offline with fast, native rendering. No internet required.

Encrypted Notes

Take notes that are encrypted with AES-256-GCM and bound to your device. Only you can read them.

DRM Protection

Multi-layer protection — native FLAG_SECURE, canvas tampering, and forensic watermarking.

Cross Platform

Available on Windows desktop and Android mobile with the same great experience.

How to install on Android
  1. Tap the Download .apk button above (or scan the QR code)
  2. Open the downloaded file when prompted

  4. Tap Install
  5. Done — open CloudShelf from your app drawer
